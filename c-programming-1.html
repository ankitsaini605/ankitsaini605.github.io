📌 What is C Programming?

C is a general-purpose, procedural programming language developed by Dennis Ritchie. It’s known for its speed and close-to-hardware performance, often used in systems and embedded programming.

📐 Basic Syntax

A basic C program has the following structure:


#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

🔢 Variables & Data Types

Common data types:

  • int – Integer
  • float – Decimal
  • char – Character

💬 Input and Output

Use scanf for input and printf for output.


int age;
printf("Enter your age: ");
scanf("%d", &age);
printf("You are %d years old", age);

✅ Simple Program Example


#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int sum = a + b;
    printf("Sum = %d", sum);
    return 0;
}
