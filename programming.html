Getting Started with Python

Python is known for its simplicity and readability, making it an excellent choice for beginners. Let's start with the classic "Hello, World!" program:

# This is a simple Python program
print("Hello, World!")

Python uses indentation to define code blocks. Here's an example of a simple conditional statement:

age = 18

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Variables and Data Types

Python has several built-in data types:

  • Integers: Whole numbers like 1, 100, -10
  • Floats: Decimal numbers like 3.14, -0.001
  • Strings: Text enclosed in quotes like "Hello" or 'Python'
  • Booleans: True or False values
  • Lists: Ordered collections of items [1, 2, 3]
  • Tuples: Immutable ordered collections (1, 2, 3)
  • Dictionaries: Key-value pairs {'name': 'John', 'age': 30}
# Examples of different data types
integer_var = 10
float_var = 3.14
string_var = "Python is awesome"
boolean_var = True
list_var = [1, 2, 3, 4, 5]
tuple_var = (1, 2, 3)
dict_var = {"name": "John", "age": 30}

Python Data Structures

Python provides several built-in data structures that are essential for efficient programming.

Lists

Lists are ordered, mutable collections that can contain items of different types:

# Creating and manipulating lists
fruits = ["apple", "banana", "cherry"]

# Accessing elements
print(fruits[0])  # Output: apple

# Adding elements
fruits.append("orange")
print(fruits)  # Output: ['apple', 'banana', 'cherry', 'orange']

# Removing elements
fruits.remove("banana")
print(fruits)  # Output: ['apple', 'cherry', 'orange']

# List comprehension
numbers = [1, 2, 3, 4, 5]
squared = [x**2 for x in numbers]
print(squared)  # Output: [1, 4, 9, 16, 25]

Dictionaries

Dictionaries store key-value pairs and provide fast lookups:

# Creating and using dictionaries
student = {
    "name": "John Smith",
    "age": 20,
    "courses": ["Math", "Science", "History"]
}

# Accessing values
print(student["name"])  # Output: John Smith

# Adding new key-value pairs
student["email"] = "john@example.com"

# Iterating through a dictionary
for key, value in student.items():
    print(key, ":", value)

Python Functions

Functions are reusable blocks of code that perform specific tasks. They help in organizing code and promoting reusability.

# Defining a simple function
def greet(name):
    return f"Hello, {name}!"

# Calling the function
message = greet("Alice")
print(message)  # Output: Hello, Alice!

# Function with default parameters
def power(base, exponent=2):
    return base ** exponent

print(power(5))      # Output: 25 (5^2)
print(power(2, 3))   # Output: 8 (2^3)

# Lambda functions (anonymous functions)
square = lambda x: x * x
print(square(4))  # Output: 16

Function Arguments

Python provides flexible ways to handle function arguments:

# *args for variable number of positional arguments
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4, 5))  # Output: 15

# **kwargs for variable number of keyword arguments
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="John", age=30, city="New York")

Object-Oriented Programming in Python

Python supports object-oriented programming (OOP) with classes and objects.

# Defining a class
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, my name is {self.name} and I am {self.age} years old."

# Creating an object
person1 = Person("Alice", 30)
print(person1.greet())

# Inheritance
class Student(Person):
    def __init__(self, name, age, student_id):
        super().__init__(name, age)
        self.student_id = student_id
    
    def study(self, subject):
        return f"{self.name} is studying {subject}."

student1 = Student("Bob", 20, "S12345")
print(student1.greet())
print(student1.study("Python"))

Special Methods

Python classes can implement special methods (also called dunder methods) to customize behavior:

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        return f"Point({self.x}, {self.y})"
    
    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)

p1 = Point(1, 2)
p2 = Point(3, 4)
p3 = p1 + p2  # Uses __add__ method
print(p3)  # Output: Point(4, 6)

Java Programming

Java is a class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It's used for developing applications for a wide range of platforms.

// A simple Java program
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java is known for its "Write Once, Run Anywhere" (WORA) capability, meaning that compiled Java code can run on all platforms that support Java without recompilation.

Key Features of Java

  • Object-Oriented: Java is based on the concept of objects, which can contain data and methods.
  • Platform Independent: Java code runs on any device that has the Java Virtual Machine (JVM).
  • Secure: Java has built-in security features designed to allow safe execution of code.
  • Robust: Java emphasizes early checking for possible errors and runtime checking.
  • Multithreaded: Java supports concurrent programming with its built-in threading capabilities.

JavaScript Programming

JavaScript is a versatile programming language primarily used for creating interactive effects within web browsers. It's an essential technology for web development.

// A simple JavaScript example
function greet(name) {
    return `Hello, ${name}!`;
}

const message = greet('World');
console.log(message);  // Output: Hello, World!

What Can JavaScript Do?

  • Change HTML content and attributes
  • Change CSS styles
  • Hide and show HTML elements
  • Validate user input before submitting to a server
  • Create cookies to store and retrieve data
  • Respond to user events like clicks, form submissions, and key presses
