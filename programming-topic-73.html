🧩 Backtracking Algorithm – Powerful Problem Solving

Backtracking is a problem-solving algorithm that uses recursion and tries to build a solution incrementally. If a solution fails, it backtracks and tries a different path. It's often used in scenarios where all permutations, combinations, or constraints need to be explored.

🔍 Characteristics of Backtracking

  • Recursive in nature
  • Follows Depth First Search (DFS)
  • Tries all possibilities and abandons those that fail
  • Used where decision trees can be applied

🎯 Common Backtracking Problems

  1. N-Queens Problem
  2. Sudoku Solver
  3. Rat in a Maze
  4. Word Search in Matrix
  5. Subset Generation
  6. Palindrome Partitioning

🧠 C++ Example: Solving N-Queens


// isSafe() function to check safety
bool isSafe(vector& board, int row, int col, int n) {
    // Check column and diagonals
    for (int i = 0; i < row; i++) {
        if (board[i][col] == 'Q') return false;
        if (col - (row - i) >= 0 && board[i][col - (row - i)] == 'Q') return false;
        if (col + (row - i) < n && board[i][col + (row - i)] == 'Q') return false;
    }
    return true;
}

// solve() function to recursively place queens
void solve(vector& board, int row, int n) {
    if (row == n) {
        // Print solution or store it
        return;
    }
    for (int col = 0; col < n; col++) {
        if (isSafe(board, row, col, n)) {
            board[row][col] = 'Q';
            solve(board, row + 1, n);
            board[row][col] = '.'; // backtrack
        }
    }
}

📌 Why Use Backtracking?

  • Explores all possibilities intelligently
  • Efficient for constraint satisfaction problems
  • Used in puzzle and game-solving algorithms
Pro Tip 🎓: Backtracking works best when you can prune paths early with smart conditions to reduce the time complexity.

🌍 Real-Life Applications

  • AI Game Playing (e.g., chess moves)
  • Word puzzles and board games (e.g., crossword, Sudoku)
  • Compiler optimization (register allocation)
  • Combinatorial problem solving in Data Science
