🔁 Palindrome Number

Check if a number is the same when reversed using Python and C++

Explore Code

📘 What is a Palindrome Number?

A number is a palindrome if it reads the same backward as forward.

Examples: 121, 1221, 1331 are all palindromes.

🧠 Python Code to Check Palindrome


def is_palindrome(num):
    return str(num) == str(num)[::-1]

number = 121
print(f"{number} is palindrome: {is_palindrome(number)}")

🔷 C++ Code to Check Palindrome


#include
using namespace std;

bool isPalindrome(int n) {
    int original = n, reversed = 0;
    while(n > 0) {
        reversed = reversed * 10 + n % 10;
        n = n / 10;
    }
    return original == reversed;
}
