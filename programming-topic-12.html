🧮 Variables and Data Types

Learn how computers store, access, and manipulate different kinds of data.

Start Learning

📘 What Are Variables?

Variables are containers for storing data values. Think of them as labels attached to data.

🔤 Variable Declaration (Python)


# Python
name = "Ankit"
age = 22
price = 199.99
is_active = True

🔤 Variable Declaration (C++)


// C++
string name = "Ankit";
int age = 22;
float price = 199.99;
bool isActive = true;

📚 Data Types

Each variable in programming has a type that defines the kind of data it holds:

Data Type Description Example
int Integer values 10, -5, 1000
float/double Decimal values 3.14, 99.99
string Text values "Hello", "EduIgnite"
bool Boolean (True/False) true, false

🔧 Best Practices

  • 💡 Always give meaningful names to variables (e.g., `userAge`, not `x`)
  • ⚠️ Use appropriate data types to save memory
  • 🧼 Keep your code readable and well-documented
💡 Tip: In dynamically typed languages like Python, you don't need to declare the data type explicitly.
