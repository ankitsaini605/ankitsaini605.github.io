🐍 Python Syntax

Python is known for its easy-to-read syntax. Here's a breakdown of the basics to get you started with writing clean and effective Python code.

1. Python Indentation

Unlike other languages, Python uses indentation (whitespace) to define blocks of code.

if 5 > 2:
    print("Five is greater than two!")

2. Python Comments

Comments start with the # symbol.

# This is a comment
print("Hello World")

3. Variables in Python

No need to declare variables with a specific type. Python figures it out for you!

x = 10
y = "Welcome to EduIgnite"
print(x, y)

4. Case-Sensitivity

Python is case-sensitive.

a = 10
A = 20
print(a)  # Outputs: 10
print(A)  # Outputs: 20

5. Multiline Strings

You can use triple quotes to write strings in multiple lines.

text = """This is
a multiline
string"""
print(text)

6. Best Practices

  • Use meaningful variable names
  • Keep code properly indented (4 spaces recommended)
  • Comment your code
  • Follow PEP8 styling guidelines
