🌐 Floyd-Warshall Algorithm

Solve the All-Pairs Shortest Path problem efficiently using Dynamic Programming.

📘 What is the Floyd-Warshall Algorithm?

It is a **Dynamic Programming-based** algorithm used to find the shortest distances between all pairs of vertices in a weighted graph (with positive or negative weights but no negative cycles).

💡 Concept:

The main idea is to iteratively update the shortest path between every pair (i, j) using an intermediate vertex k.

⚙️ Python Implementation:


INF = float('inf')

def floyd_warshall(graph):
    V = len(graph)
    dist = list(map(lambda row: list(row), graph))

    for k in range(V):
        for i in range(V):
            for j in range(V):
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]
    
    return dist

graph = [
    [0, 3, INF, 5],
    [2, 0, INF, 4],
    [INF, 1, 0, INF],
    [INF, INF, 2, 0]
]

result = floyd_warshall(graph)
for row in result:
    print(row)

🔧 C++ Code:


#include <iostream>
using namespace std;
#define INF 99999
#define V 4

void floydWarshall(int graph[V][V]) {
    int dist[V][V];

    for (int i = 0; i < V; i++)
        for (int j = 0; j < V; j++)
            dist[i][j] = graph[i][j];

    for (int k = 0; k < V; k++) {
        for (int i = 0; i < V; i++) {
            for (int j = 0; j < V; j++) {
                if (dist[i][k] + dist[k][j] < dist[i][j])
                    dist[i][j] = dist[i][k] + dist[k][j];
            }
        }
    }

    for (int i = 0; i < V; i++) {
        for (int j = 0; j < V; j++) {
            if (dist[i][j] == INF)
                cout << "INF ";
            else
                cout << dist[i][j] << " ";
        }
        cout << endl;
    }
}

int main() {
    int graph[V][V] = {
        {0, 5, INF, 10},
        {INF, 0, 3, INF},
        {INF, INF, 0,   1},
        {INF, INF, INF, 0}
    };
    floydWarshall(graph);
    return 0;
}

⏱️ Time Complexity:

  • O(V³), where V is the number of vertices.

✅ Use Cases:

  • Finding the shortest routes between every pair of cities
  • Routing in computer networks
  • Dynamic programming-based graph applications
