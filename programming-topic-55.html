🔁 Deque (Double Ended Queue)

Learn everything about Deques in Data Structures with implementation in C++ and Python.

📘 What is a Deque?

A Deque (Double-Ended Queue) is a data structure where elements can be added or removed from both ends (front and rear). It supports both FIFO and LIFO operations.

🔧 Operations on Deque:

  • push_front() – Add element at the front
  • push_back() – Add element at the back
  • pop_front() – Remove from front
  • pop_back() – Remove from back
  • front() and back() – View elements

💻 Python Implementation:


from collections import deque

dq = deque()

# Insert at both ends
dq.append(10)
dq.appendleft(5)

# Remove from both ends
dq.pop()
dq.popleft()

print("Deque:", list(dq))

💻 C++ Implementation:


#include <iostream>
#include <deque>
using namespace std;

int main() {
    deque<int> dq;

    dq.push_back(10);
    dq.push_front(5);

    dq.pop_back();
    dq.pop_front();

    cout << "Deque size: " << dq.size() << endl;
    return 0;
}

🔍 Applications of Deque:

  • Browser history (back and forward buttons)
  • Sliding window problems
  • Palindrome checking
  • Scheduling and job queues

🧠 Time Complexity:

  • Insertion & Deletion (both ends): O(1)
  • Access (front/back): O(1)

📌 Difference from Queue & Stack:

  • Queue: Only insert rear, delete front
  • Stack: Only insert/delete from one end
  • Deque: Insert/delete from both ends
