1. Which of the following is the correct syntax of main function in C?

✅ Correct: int main()

2. What is the output of: printf("%d", (5 + 2) * 3);

✅ Correct: 21
Explanation: (5 + 2) = 7, then 7 * 3 = 21

3. Which symbol is used to denote a pointer?

✅ Correct: *
Explanation: Asterisk (*) is used to declare and dereference pointers in C.
