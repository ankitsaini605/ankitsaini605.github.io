C++ Essentials – Functions, Loops, Arrays

🧩 Functions

int add(int a, int b) {
  return a + b;
}
int main() {
  cout << add(5, 3); // Output: 8
}

🔁 Loops (for)

for(int i = 1; i <= 5; i++) {
  cout << i << " ";
}
// Output: 1 2 3 4 5

📊 Arrays

int arr[3] = {10, 20, 30};
for(int i = 0; i < 3; i++) {
  cout << arr[i] << " ";
}
// Output: 10 20 30
