Variables & Data Types

Understand how computers store and use different types of information using variables.

Explore Now

🧠 What Are Variables?

Variables are named memory locations used to store data. Think of them as labeled containers where you can keep values like numbers, text, or more complex data.

Declaring Variables

x = 10
name = "Ankit"
pi = 3.14

Above are examples of integer, string, and float variables in Python. Each programming language has different syntax but the logic remains similar.

📦 Data Types

Each variable in programming holds a specific type of data. Common data types include:

Data Type Description Example
Integer (int) Whole numbers 5, 100, -3
Float (float) Decimal numbers 3.14, -0.99
String (str) Text "Hello", "EduIgnite"
Boolean (bool) True/False True, False

💡 Type Conversion

Sometimes we need to convert one data type to another. This is known as type casting.


x = "100"
y = int(x)  # y becomes integer 100
