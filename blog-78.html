📌 Dynamic Programming (DP)

Dynamic Programming is an optimization technique used to solve complex problems by breaking them down into simpler overlapping sub-problems and solving each only once, storing their results for future reference.

⚙️ Key Concepts

  • Overlapping Subproblems
  • Optimal Substructure
  • Memoization (Top-Down)
  • Tabulation (Bottom-Up)

📊 Common Applications

  • Fibonacci Numbers
  • 0/1 Knapsack Problem
  • Longest Common Subsequence
  • Matrix Chain Multiplication
  • Rod Cutting Problem

💻 Code Example: Fibonacci (Memoization)


int fib(int n, vector<int>& dp) {
  if (n <= 1) return n;
  if (dp[n] != -1) return dp[n];
  return dp[n] = fib(n-1, dp) + fib(n-2, dp);
}

// Driver Code
int n = 10;
vector<int> dp(n+1, -1);
cout << fib(n, dp);

📈 Time Complexity

  • Naive Recursion: O(2^n)
  • DP (Memoized): O(n)

✅ Real-World Benefits

  • Efficient solution to exponential problems
  • Reduces redundant computations
  • Used in AI, operations research, bioinformatics, and more
