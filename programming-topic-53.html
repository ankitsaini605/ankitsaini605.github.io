📚 Stack - Data Structure

Last-In, First-Out (LIFO) structure to manage elements effectively

📘 What is a Stack?

A Stack is a linear data structure that follows the LIFO (Last In First Out) principle. The last element added to the stack is the first one to be removed.

🔧 Basic Operations:

  • push() – Add an item to the stack
  • pop() – Remove the top item
  • peek() / top() – View the top item
  • isEmpty() – Check if the stack is empty

💻 Python Example:


stack = []

# Push
stack.append(10)
stack.append(20)
stack.append(30)

# Pop
print("Popped:", stack.pop())

# Top Element
print("Top:", stack[-1])

# Check if Empty
print("Is Empty:", len(stack) == 0)

💻 C++ Example:


#include <iostream>
#include <stack>
using namespace std;

int main() {
    stack<int> s;
    s.push(10);
    s.push(20);
    s.push(30);

    cout << "Top: " << s.top() << endl;
    s.pop();
    cout << "After pop, Top: " << s.top() << endl;

    cout << "Is Empty: " << s.empty() << endl;
    return 0;
}

📦 Stack Applications:

  • Function call management (call stack)
  • Undo features in editors
  • Syntax parsing
  • Expression evaluation (Postfix, Prefix)

🧮 Time Complexity:

  • Push: O(1)
  • Pop: O(1)
  • Peek/Top: O(1)

📌 Stack VS Queue:

  • Stack: LIFO (Last-In-First-Out)
  • Queue: FIFO (First-In-First-Out)
