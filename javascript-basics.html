📜 JavaScript Basics

🔍 What is JavaScript?

JavaScript is a powerful scripting language used to create dynamic and interactive experiences on web pages. It runs in the browser and allows developers to control HTML content, validate forms, create animations, and more.

🧠 Why Learn JavaScript?

  • Essential for front-end web development
  • Works with HTML and CSS
  • Used in all modern websites & web apps
  • Can also be used on the server-side (Node.js)

💡 JavaScript Syntax Example

// Output a message
console.log("Hello from EduIgnite!");

// Variables
let name = "Ankit";
const age = 22;

// Functions
function greet(user) {
  return "Welcome, " + user + "!";
}

console.log(greet(name));
✅ Tip: Use let for variables and const for constants in modern JS.
