🔁 Recursion in Programming

A powerful concept where a function calls itself to solve smaller instances of a problem.

Explore Recursion

📘 What is Recursion?

Recursion is a technique in programming where a function calls itself repeatedly until a base condition is met. It is especially useful for problems that can be divided into similar sub-problems.

🧠 Structure of a Recursive Function

  • 🔹 A base case to stop recursion.
  • 🔹 A recursive case where the function calls itself.

📍 Example: Factorial (Python)


def factorial(n):
    if n == 0 or n == 1:  # Base Case
        return 1
    else:
        return n * factorial(n - 1)  # Recursive Case

print(factorial(5))  # Output: 120

📍 Example: Factorial (C++)


int factorial(int n) {
    if (n == 0 || n == 1)
        return 1; // Base case
    else
        return n * factorial(n - 1); // Recursive call
}

🔎 Where is Recursion Used?

  • Solving mathematical problems (factorial, Fibonacci, GCD)
  • Tree traversals (Inorder, Preorder, Postorder)
  • Backtracking (Maze solving, N-Queens problem)
  • Divide and conquer algorithms (Merge Sort, Quick Sort)

⚠️ Drawbacks

  • Consumes more memory due to multiple function calls on the stack
  • Can cause stack overflow if base case is not handled correctly
  • In some cases, iterative solutions are more efficient
💡 Tip: Always write the base case first to prevent infinite recursion.
