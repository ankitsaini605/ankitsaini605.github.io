⚡ Quick Sort Algorithm

One of the fastest and most efficient divide-and-conquer sorting algorithms used in software systems.

📚 Introduction

Quick Sort is a **divide-and-conquer** algorithm. It picks an element as a pivot and partitions the array around the pivot such that:

  • All elements on the left are smaller
  • All elements on the right are larger

📌 Real-World Use Cases:

  • Databases & search engines
  • Large-scale enterprise software
  • Gaming engines for sorting scores & levels

🐍 Python Code


def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr)//2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [10, 7, 8, 9, 1, 5]
print("Sorted Array:", quick_sort(arr))

💻 C++ Code


#include <iostream>
using namespace std;

int partition(int arr[], int low, int high) {
    int pivot = arr[high];
    int i = (low - 1);
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return (i + 1);
}

void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

int main() {
    int arr[] = {10, 7, 8, 9, 1, 5};
    int n = sizeof(arr)/sizeof(arr[0]);
    quickSort(arr, 0, n - 1);
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) cout << arr[i] << " ";
    return 0;
}

📈 Complexity Analysis

  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n²) — when pivot is smallest/largest every time
  • Space: O(log n)

✅ Advantages

  • Efficient for large datasets
  • In-place sorting (no extra memory needed)

❌ Disadvantages

  • Not stable by default
  • Worst-case performance is poor (can be fixed with randomized pivot)
