🎒 0/1 Knapsack Problem

Understand and solve one of the most popular dynamic programming problems with optimized techniques.

Start Solving

📘 Problem Statement

You are given weights and values of N items, put these items in a knapsack of capacity W to get the maximum total value in the knapsack. You cannot break an item, either pick it or don’t (0/1 property).

🧠 Dynamic Programming Approach

We use a 2D array dp[n+1][W+1] where each dp[i][w] represents the maximum value for first i items with weight limit w.

🧪 Python Code


def knapsack(weights, values, W):
    n = len(weights)
    dp = [[0]*(W+1) for _ in range(n+1)]

    for i in range(1, n+1):
        for w in range(W+1):
            if weights[i-1] <= w:
                dp[i][w] = max(values[i-1] + dp[i-1][w - weights[i-1]], dp[i-1][w])
            else:
                dp[i][w] = dp[i-1][w]
    return dp[n][W]

# Example usage:
weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
capacity = 5
print(knapsack(weights, values, capacity))  # Output: 7

using namespace std;

int knapsack(int W, vector wt, vector val, int n) {
    vector> dp(n+1, vector(W+1, 0));
    
    for (int i = 1; i <= n; ++i) {
        for (int w = 0; w <= W; ++w) {
            if (wt[i-1] <= w)
                dp[i][w] = max(val[i-1] + dp[i-1][w - wt[i-1]], dp[i-1][w]);
            else
                dp[i][w] = dp[i-1][w];
        }
    }
    return dp[n][W];
}

int main() {
    vector val = {3, 4, 5, 6};
    vector wt = {2, 3, 4, 5};
    int W = 5;
    cout << knapsack(W, wt, val, val.size()) << endl;  // Output: 7
    return 0;
}

🔍 Optimization Tips

  • Space optimization: use 1D array instead of 2D.
  • Greedy doesn’t work — must use DP for 0/1 knapsack.
