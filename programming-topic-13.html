📘 What is the Fibonacci Series?

The Fibonacci sequence is a series where each number is the sum of the two preceding ones. It starts from 0 and 1:

0, 1, 1, 2, 3, 5, 8, 13, ...

🧮 Recursive Solution (Python)


def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)

print(fib(6))  # Output: 8

⚡ Memoization (Top-Down DP)


def fib_memo(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fib_memo(n-1, memo) + fib_memo(n-2, memo)
    return memo[n]

print(fib_memo(50))  # Fast output

📊 Tabulation (Bottom-Up DP)


def fib_tab(n):
    if n <= 1:
        return n
    dp = [0, 1]
    for i in range(2, n+1):
        dp.append(dp[i-1] + dp[i-2])
    return dp[n]

print(fib_tab(10))  # Output: 55

💡 Fibonacci in C++ (Bottom-Up)


#include

using namespace std;

int fib(int n) {
    if (n <= 1) return n;
    vector dp(n+1);
    dp[0] = 0;
    dp[1] = 1;
    for(int i = 2; i <= n; i++)
        dp[i] = dp[i-1] + dp[i-2];
    return dp[n];
}

int main() {
    cout << fib(10);  // Output: 55
    return 0;
}
