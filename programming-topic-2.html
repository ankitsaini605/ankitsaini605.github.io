🔍 What is Even or Odd?

A number is even if it's divisible by 2; otherwise, it's odd. For example:

  • Even: 2, 4, 6, 8...
  • Odd: 1, 3, 5, 7...

✅ Python Code


num = int(input("Enter a number: "))

if num % 2 == 0:
    print(num, "is Even")
else:
    print(num, "is Odd")

⚙️ C++ Code


#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "Enter a number: ";
    cin >> num;

    if (num % 2 == 0)
        cout << num << " is Even" << endl;
    else
        cout << num << " is Odd" << endl;

    return 0;
}

📝 Note:

This program uses the modulus operator % to find the remainder. If num % 2 == 0, the number is even.
