📘 What is the Fibonacci Series?

The Fibonacci Series is a series of numbers where each number is the sum of the two preceding ones.

Sequence: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

🧠 Python Code: Fibonacci Using Loop


def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        print(a, end=" ")
        a, b = b, a + b

fibonacci(10)

🔁 Python Code: Fibonacci Using Recursion


def fib_recursive(n):
    if n <= 1:
        return n
    return fib_recursive(n-1) + fib_recursive(n-2)

for i in range(10):
    print(fib_recursive(i), end=" ")

🔷 C++ Code: Fibonacci Using Loop


#include <iostream>
using namespace std;

void fibonacci(int n) {
    int a = 0, b = 1, c;
    for (int i = 0; i < n; i++) {
        cout << a << " ";
        c = a + b;
        a = b;
        b = c;
    }
}

int main() {
    fibonacci(10);
    return 0;
}

🧾 Explanation

  • Loop Method: Efficient for generating series quickly.
  • Recursion: Useful for understanding function calls, but slow for large n.
  • Both codes follow clean and readable structures, ideal for beginners.
