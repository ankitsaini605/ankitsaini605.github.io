Time and Work Aptitude Problems

Time and Work problems are a key component of quantitative aptitude tests used in various competitive exams and interviews. Mastering these questions will improve your problem-solving speed and accuracy.

Basic Concepts

  • Work Done: Usually considered as 1 unit (total work).
  • Work Rate: Work done per unit time.
  • Time: Time taken to complete the work.
  • Formula: Work = Rate × Time
  • If A can do a work in a days and B in b days, then combined work rate = 1/a + 1/b

Important Formulas

  • Work done in one day by A = 1 / a
  • Work done in one day by B = 1 / b
  • Work done together in one day = 1/a + 1/b
  • Time taken together = 1 / (1/a + 1/b)

Sample Problems with Solutions

Problem 1:

A can complete a work in 12 days and B in 15 days. How long will it take for both working together to complete the work?

Solution:

Work rate of A = 1/12, work rate of B = 1/15

Combined work rate = 1/12 + 1/15 = (5 + 4) / 60 = 9/60 = 3/20

Time taken = 1 / (3/20) = 20/3 = 6.67 days

Problem 2:

If A alone can do a work in 10 days, B alone in 20 days, and C alone in 30 days. Find the time taken if all three work together.

Solution:

Combined work rate = 1/10 + 1/20 + 1/30 = (6 + 3 + 2) / 60 = 11/60

Time taken = 1 / (11/60) = 60/11 ≈ 5.45 days

Problem 3:

A is twice as efficient as B. If together they can complete a work in 12 days, how long will B alone take?

Solution:

Let B's rate be r, then A's rate = 2r

Combined rate = r + 2r = 3r

Given 3r = 1/12 ⇒ r = 1/36

Time taken by B alone = 1 / r = 36 days

Tips to Solve Time and Work Problems

  • Always convert the work done into a single unit (like 1 complete work).
  • Use the concept of work rate per day or hour.
  • Understand if workers are working together or separately.
  • Remember the relation between efficiency and work rate.
  • Practice problems regularly to increase speed and accuracy.
