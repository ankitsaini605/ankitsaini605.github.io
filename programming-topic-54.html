📚 Queue - Data Structure

Understand how Queues work with FIFO logic using code examples and practical applications.

📘 What is a Queue?

A Queue is a linear data structure that follows the FIFO (First In First Out) principle. It works just like a queue in real life, where the first person in line is the first to be served.

🔧 Basic Operations:

  • enqueue() – Add item to the rear
  • dequeue() – Remove item from the front
  • front() – Get the front item
  • isEmpty() – Check if the queue is empty

💻 Python Example:


from collections import deque

queue = deque()

# Enqueue
queue.append(10)
queue.append(20)
queue.append(30)

# Dequeue
print("Dequeued:", queue.popleft())

# Front Element
print("Front:", queue[0])

# Check if Empty
print("Is Empty:", not queue)

💻 C++ Example:


#include <iostream>
#include <queue>
using namespace std;

int main() {
    queue<int> q;

    q.push(10);
    q.push(20);
    q.push(30);

    cout << "Front: " << q.front() << endl;
    q.pop();
    cout << "After pop, Front: " << q.front() << endl;

    cout << "Is Empty: " << q.empty() << endl;
    return 0;
}

📦 Queue Applications:

  • Task scheduling (OS)
  • Print queues
  • Data buffering (IO Buffers)
  • Breadth-First Search (BFS)

🧮 Time Complexity:

  • Enqueue: O(1)
  • Dequeue: O(1)
  • Front: O(1)

🆚 Stack vs Queue:

  • Stack: LIFO – Last-In-First-Out
  • Queue: FIFO – First-In-First-Out
