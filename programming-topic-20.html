📈 Factorial of a Number

Learn how to calculate factorial of any number using Python and C++ step-by-step.

Jump to Code

💡 What is a Factorial?

The factorial of a number n is the product of all positive integers less than or equal to n. It is denoted by n!.

Example: 5! = 5 × 4 × 3 × 2 × 1 = 120

✅ Python Code (Using Loop)


num = int(input("Enter a number: "))
factorial = 1

if num < 0:
    print("Factorial does not exist for negative numbers.")
elif num == 0:
    print("The factorial of 0 is 1.")
else:
    for i in range(1, num + 1):
        factorial *= i
    print("The factorial of", num, "is", factorial)

⚙️ C++ Code (Using Loop)


#include <iostream>
using namespace std;

int main() {
    int num;
    long long factorial = 1;

    cout << "Enter a number: ";
    cin >> num;

    if (num < 0)
        cout << "Factorial does not exist for negative numbers." << endl;
    else {
        for (int i = 1; i <= num; ++i) {
            factorial *= i;
        }
        cout << "Factorial of " << num << " = " << factorial << endl;
    }

    return 0;
}

📘 Tip:

Factorials grow very fast! 20! is already a number with 19 digits. Use long long or BigInteger types for large inputs.
