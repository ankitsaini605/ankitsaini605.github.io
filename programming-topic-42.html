⚡ Binary Search Algorithm

Explore the efficient Binary Search algorithm with step-by-step C++ and Python code examples, visual explanation, and time complexity breakdown.

📘 What is Binary Search?

Binary Search is a powerful searching technique used on sorted arrays. It repeatedly divides the search interval in half, making it significantly faster than linear search for large datasets.

🧠 Requirements:

  • Array must be sorted in ascending or descending order
  • Efficient when dealing with large datasets

🐍 Python Code Example


def binary_search(arr, key):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == key:
            return mid
        elif arr[mid] < key:
            low = mid + 1
        else:
            high = mid - 1
    return -1

data = [5, 10, 15, 20, 25, 30]
key = int(input("Enter element to search: "))
index = binary_search(data, key)
if index != -1:
    print(f"Element found at index {index}")
else:
    print("Element not found")

💻 C++ Code Example


#include <iostream>
using namespace std;

int binarySearch(int arr[], int size, int key) {
    int low = 0, high = size - 1;
    while (low <= high) {
        int mid = (low + high) / 2;
        if (arr[mid] == key)
            return mid;
        else if (arr[mid] < key)
            low = mid + 1;
        else
            high = mid - 1;
    }
    return -1;
}

int main() {
    int arr[] = {5, 10, 15, 20, 25, 30};
    int key;
    cout << "Enter key to search: ";
    cin >> key;
    int result = binarySearch(arr, 6, key);
    if (result != -1)
        cout << "Element found at index " << result;
    else
        cout << "Element not found";
    return 0;
}

📊 Time Complexity:

  • Best Case: O(1) — found at mid
  • Average/Worst Case: O(log n)

⚠️ Common Mistakes:

  • Using it on unsorted arrays
  • Forgetting to update `low` or `high` in loop
