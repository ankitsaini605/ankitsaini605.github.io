🌐 Graph Algorithms

Graphs are a fundamental part of computer science. They model networks, dependencies, and more. Mastering graph algorithms is essential for cracking competitive coding and interviews.

📌 Common Graph Representations

  • Adjacency Matrix
  • Adjacency List
  • Edge List

🔍 Breadth First Search (BFS)


void bfs(int start, vector<int> adj[], int V) {
  vector<bool> visited(V, false);
  queue<int> q;

  visited[start] = true;
  q.push(start);

  while (!q.empty()) {
    int node = q.front();
    q.pop();
    cout << node << " ";

    for (int neighbor : adj[node]) {
      if (!visited[neighbor]) {
        visited[neighbor] = true;
        q.push(neighbor);
      }
    }
  }
}

🧭 Depth First Search (DFS)


void dfs(int node, vector<int> adj[], vector<bool> &visited) {
  visited[node] = true;
  cout << node << " ";

  for (int neighbor : adj[node]) {
    if (!visited[neighbor]) {
      dfs(neighbor, adj, visited);
    }
  }
}

🚦 Dijkstra’s Algorithm

Finds the shortest path from a source to all vertices in a weighted graph.


void dijkstra(int V, vector<pair<int,int>> adj[], int src) {
  vector<int> dist(V, INT_MAX);
  dist[src] = 0;
  priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;

  pq.push({0, src});

  while (!pq.empty()) {
    int node = pq.top().second;
    int d = pq.top().first;
    pq.pop();

    for (auto edge : adj[node]) {
      int next = edge.first;
      int weight = edge.second;

      if (d + weight < dist[next]) {
        dist[next] = d + weight;
        pq.push({dist[next], next});
      }
    }
  }
}

🧠 Other Key Algorithms

  • Topological Sort
  • Prim’s & Kruskal’s (Minimum Spanning Tree)
  • Bellman-Ford Algorithm
  • Floyd Warshall Algorithm
  • Union-Find (Disjoint Sets)

📈 Time Complexities

  • BFS & DFS: O(V + E)
  • Dijkstra (with Min Heap): O(E log V)
  • Bellman-Ford: O(VE)
  • Floyd-Warshall: O(V³)
Tip: Always choose the right graph representation (List vs Matrix) based on density and operations.
