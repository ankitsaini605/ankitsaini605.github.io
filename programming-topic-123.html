🔢 Fibonacci Series in Programming

Learn how to generate the Fibonacci Series using recursion and iteration in Python and C++.

Explore Now

📘 What is the Fibonacci Series?

The Fibonacci Series is a sequence where each number is the sum of the two preceding ones, usually starting with 0 and 1. Example: 0, 1, 1, 2, 3, 5, 8, 13...

🧪 Recursive Approach (Python)


def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# Print the first 10 Fibonacci numbers
for i in range(10):
    print(fibonacci(i), end=" ")

🧪 Iterative Approach (Python)


def fibonacci_iterative(n):
    a, b = 0, 1
    for i in range(n):
        print(a, end=" ")
        a, b = b, a + b

fibonacci_iterative(10)

🧪 Recursive Approach (C++)


int fibonacci(int n) {
    if(n <= 1)
        return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

for(int i = 0; i < 10; i++)
    cout << fibonacci(i) << " ";

🧪 Iterative Approach (C++)


void fibonacci_iterative(int n) {
    int a = 0, b = 1;
    for(int i = 0; i < n; i++) {
        cout << a << " ";
        int next = a + b;
        a = b;
        b = next;
    }
}

📈 Time Complexity

  • 🔁 Iterative: O(n)
  • 🔄 Recursive: O(2n) (Exponential without memoization)
