🔐 OOP Concepts: Inheritance & Encapsulation

🧬 Inheritance

class Animal {
public:
  void speak() { cout << "Sound"; }
};

class Dog : public Animal {
public:
  void bark() { cout << "Bark"; }
};

int main() {
  Dog d;
  d.speak();  // Output: Sound
  d.bark();   // Output: Bark
}

🛡️ Encapsulation

class Student {
private:
  int age;
public:
  void setAge(int a) { age = a; }
  int getAge() { return age; }
};

Encapsulation = Wrapping data using private members
