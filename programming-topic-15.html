🧮 Factorial Using Recursion

Explore how to calculate factorials using recursive methods in Python and C++.

Start Learning

📘 What is Factorial?

Factorial of a non-negative integer n is the product of all positive integers less than or equal to n. It's denoted as n!.

Example: 5! = 5 × 4 × 3 × 2 × 1 = 120

🔁 Factorial Using Recursion in Python


def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # Output: 120

🔷 Factorial Using Recursion in C++


#include
using namespace std;

int factorial(int n) {
    if(n == 0 || n == 1)
        return 1;
    return n * factorial(n - 1);
}

int main() {
    cout << factorial(5); // Output: 120
    return 0;
}

⚠️ Things to Remember

  • Recursive functions must have a base case to stop the recursion.
  • Be cautious of very large values of n, which can lead to stack overflow.
  • Factorials grow rapidly: 10! = 3,628,800
