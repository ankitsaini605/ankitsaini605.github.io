🔗 Floyd-Warshall Algorithm

Find shortest paths between all pairs of vertices in a weighted graph (including negative edges).

📘 Overview

The Floyd-Warshall algorithm is a dynamic programming method used to compute the shortest paths between all pairs of vertices in a graph. It works with positive and negative edge weights, but no negative cycles.

🛠️ Key Concepts:

  • Dynamic programming approach
  • Works for directed & undirected graphs
  • Handles negative edges (not cycles)
  • Time Complexity: O(V³)

💻 Python Code:


INF = float('inf')

def floyd_warshall(graph):
    V = len(graph)
    dist = [row[:] for row in graph]  # Deep copy

    for k in range(V):
        for i in range(V):
            for j in range(V):
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]

    # Print shortest distance matrix
    for row in dist:
        print(" ".join(f"{x if x != INF else 'INF':>5}" for x in row))

# Example graph (Adjacency Matrix)
graph = [
    [0,   3,   INF, 5],
    [2,   0,   INF, 4],
    [INF, 1,   0,   INF],
    [INF, INF, 2,   0]
]

floyd_warshall(graph)

💻 C++ Code:


#include <iostream>
#include <vector>
#define INF 99999
using namespace std;

void floydWarshall(vector<vector<int>> &graph) {
    int V = graph.size();
    vector<vector<int>> dist = graph;

    for (int k = 0; k < V; k++)
        for (int i = 0; i < V; i++)
            for (int j = 0; j < V; j++)
                if (dist[i][k] + dist[k][j] < dist[i][j])
                    dist[i][j] = dist[i][k] + dist[k][j];

    for (int i = 0; i < V; i++) {
        for (int j = 0; j < V; j++) {
            if (dist[i][j] == INF)
                cout << "INF ";
            else
                cout << dist[i][j] << " ";
        }
        cout << endl;
    }
}

int main() {
    vector<vector<int>> graph = {
        {0,   3,   INF, 5},
        {2,   0,   INF, 4},
        {INF, 1,   0,   INF},
        {INF, INF, 2,   0}
    };
    floydWarshall(graph);
    return 0;
}

📌 Applications:

  • Network routing protocols (OSPF)
  • Flight booking systems
  • Social networks for shortest relationship path
  • Distance matrix for logistics optimization

📈 Pros & Cons:

  • ✅ Pros: Simple, works on all-pairs, handles negative weights
  • ⚠️ Cons: Time complexity O(V³) is high for large graphs

📖 Real-Life Example:

Imagine a delivery service with many warehouse locations. The Floyd-Warshall algorithm helps compute the fastest delivery paths between all combinations of warehouses and cities.
