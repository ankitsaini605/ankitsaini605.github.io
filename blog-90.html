🧠 Backtracking Algorithms

Backtracking is a general algorithmic technique that tries out all possible solutions and abandons a path as soon as it determines that the path cannot lead to a valid solution.

How it Works:

  1. Choose - Make a choice
  2. Explore - Move forward with the choice
  3. Unchoose - If it leads to failure, undo and try another option

Popular Backtracking Problems:

  • N-Queens Problem
  • Sudoku Solver
  • Rat in a Maze
  • Permutations & Combinations
  • Word Search in Matrix

🔥 Example: Solving N-Queens Problem


bool isSafe(vector<string> &board, int row, int col, int n) {
  for (int i = 0; i < row; i++) {
    if (board[i][col] == 'Q') return false;
    if (col - (row - i) >= 0 && board[i][col - (row - i)] == 'Q') return false;
    if (col + (row - i) < n && board[i][col + (row - i)] == 'Q') return false;
  }
  return true;
}

void solve(int row, int n, vector<string>& board, vector<vector<string>>& ans) {
  if (row == n) {
    ans.push_back(board);
    return;
  }

  for (int col = 0; col < n; col++) {
    if (isSafe(board, row, col, n)) {
      board[row][col] = 'Q';
      solve(row + 1, n, board, ans);
      board[row][col] = '.'; // Backtrack
    }
  }
}

Real-life Application

Backtracking is widely used in game-solving, puzzle generation, configuration problems, and even AI logic-based systems.

Tips for Backtracking

  • Use recursion smartly
  • Always backtrack after a recursive call
  • Use pruning techniques (like forward checking)
