📘 What is Linear Search?

Linear Search is the simplest searching algorithm. It checks each element one by one until it finds the target or reaches the end of the array.

Used when:

  • The array is small or unsorted
  • Fast coding is more important than performance

✅ Python Code Example


def linear_search(arr, key):
    for i in range(len(arr)):
        if arr[i] == key:
            return i
    return -1

data = [10, 25, 30, 45, 50]
key = int(input("Enter number to search: "))
result = linear_search(data, key)

if result != -1:
    print(f"Element found at index {result}")
else:
    print("Element not found")

⚙️ C++ Code Example


#include <iostream>
using namespace std;

int linearSearch(int arr[], int size, int key) {
    for (int i = 0; i < size; i++) {
        if (arr[i] == key) return i;
    }
    return -1;
}

int main() {
    int arr[] = {10, 25, 30, 45, 50};
    int key;
    cout << "Enter number to search: ";
    cin >> key;
    int index = linearSearch(arr, 5, key);

    if (index != -1)
        cout << "Element found at index " << index << endl;
    else
        cout << "Element not found" << endl;

    return 0;
}

📊 Time Complexity

  • Best Case: O(1) — element found at first index
  • Average Case: O(n)
  • Worst Case: O(n) — element not present

💡 Tips for Interviews:

  • Don't forget to mention edge cases like empty arrays.
  • Know when to use Binary Search instead (sorted array only).
  • Be ready to explain time complexity clearly.
