📘 What is a Prime Number?

A prime number is a number greater than 1 that has no positive divisors other than 1 and itself. For example: 2, 3, 5, 7, 11, 13, ...

🔁 Basic Prime Check in Python


def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True

print(is_prime(11))  # Output: True

⚡ Optimized Prime Check (√n logic)


import math

def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(math.sqrt(n)) + 1):
        if n % i == 0:
            return False
    return True

print(is_prime(97))  # Output: True

using namespace std;

bool isPrime(int n) {
    if(n <= 1) return false;
    for(int i = 2; i <= sqrt(n); i++) {
        if(n % i == 0)
            return false;
    }
    return true;
}

int main() {
    cout << isPrime(29); // Output: 1 (true)
    return 0;
}

📌 Notes

  • Use √n optimization for faster results.
  • Prime number check is essential for cryptography, algorithms, and competitive programming.
  • Time Complexity (Optimized): O(√n)
