🧮 Binary Heap in DSA

Understand the working of Min Heap and Max Heap with practical code examples and use cases.

🔍 What is a Binary Heap?

A Binary Heap is a complete binary tree that satisfies the heap property:

  • Min Heap: Parent node ≤ child nodes
  • Max Heap: Parent node ≥ child nodes

🧰 Applications:

  • Implementing Priority Queues
  • Dijkstra’s Algorithm
  • Heap Sort
  • Real-time scheduling systems

💻 Python Min Heap Example:


import heapq

min_heap = []
heapq.heappush(min_heap, 10)
heapq.heappush(min_heap, 5)
heapq.heappush(min_heap, 20)

print("Min element:", heapq.heappop(min_heap))

💻 C++ Max Heap Example:


#include <iostream>
#include <queue>
using namespace std;

int main() {
    priority_queue<int> maxHeap;

    maxHeap.push(10);
    maxHeap.push(30);
    maxHeap.push(20);

    cout << "Max element: " << maxHeap.top() << endl;
    return 0;
}

🛠️ Heap Operations & Time Complexities

  • Insert: O(log n)
  • Delete (Extract Root): O(log n)
  • Peek: O(1)

🔄 Binary Heap vs Binary Tree:

Binary Heap Binary Tree
Complete tree Can be incomplete
Follows heap property No such restriction
