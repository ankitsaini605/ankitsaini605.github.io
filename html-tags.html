🔥 HTML Tags You Must Know

HTML (HyperText Markup Language) uses tags to structure content on the web. Below are some of the most important tags explained with examples:

1. Headings

<h1>Main Heading</h1>
<h2>Subheading</h2>

2. Paragraph

<p>This is a paragraph of text.</p>

3. Links

<a href="https://www.eduignite.com">Visit EduIgnite</a>

4. Images

<img src="logo.png" alt="EduIgnite Logo">

5. Lists

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

6. Tables

<table>
  <tr><th>Name</th><th>Marks</th></tr>
  <tr><td>Ankit</td><td>95</td></tr>
</table>

7. Forms

<form action="/submit" method="post">
  <input type="text" name="username" placeholder="Enter Name">
  <input type="submit">
</form>

8. Semantic Tags

<header>, <footer>, <section>, <article>, <nav>, <main>

🔔 Pro Tip

Use semantic tags for better accessibility and SEO!
