📍 Bellman-Ford Algorithm

Handle graphs with negative weights and detect negative cycles effectively.

🔍 What is Bellman-Ford Algorithm?

The Bellman-Ford algorithm is used to compute the shortest paths from a single source vertex to all other vertices in a weighted graph. Unlike Dijkstra, it works with negative weight edges.

📘 Key Features:

  • Supports graphs with negative weight edges
  • Detects negative weight cycles
  • Time Complexity: O(V * E)

💡 Algorithm Steps:

  1. Initialize distance to source = 0, and others = ∞
  2. Relax all edges (V-1) times
  3. Check for negative weight cycles in 1 final iteration

💻 Python Code:


def bellman_ford(edges, V, src):
    dist = [float('inf')] * V
    dist[src] = 0

    for _ in range(V - 1):
        for u, v, wt in edges:
            if dist[u] != float('inf') and dist[u] + wt < dist[v]:
                dist[v] = dist[u] + wt

    # Check for negative weight cycle
    for u, v, wt in edges:
        if dist[u] != float('inf') and dist[u] + wt < dist[v]:
            print("Graph contains negative weight cycle")
            return

    print("Vertex Distance from Source:", dist)

# Example usage
edges = [(0, 1, 4), (0, 2, 5), (1, 2, -3), (2, 3, 4)]
V = 4
bellman_ford(edges, V, 0)

💻 C++ Code:


#include <iostream>
#include <vector>
#include <climits>
using namespace std;

struct Edge {
    int src, dest, weight;
};

void bellmanFord(vector<Edge>& edges, int V, int src) {
    vector<int> dist(V, INT_MAX);
    dist[src] = 0;

    for (int i = 1; i < V; ++i) {
        for (auto &e : edges) {
            if (dist[e.src] != INT_MAX && dist[e.src] + e.weight < dist[e.dest]) {
                dist[e.dest] = dist[e.src] + e.weight;
            }
        }
    }

    // Check for negative weight cycles
    for (auto &e : edges) {
        if (dist[e.src] != INT_MAX && dist[e.src] + e.weight < dist[e.dest]) {
            cout << "Graph contains negative weight cycle\n";
            return;
        }
    }

    for (int i = 0; i < V; ++i)
        cout << "Distance from " << src << " to " << i << " is " << dist[i] << endl;
}

int main() {
    int V = 4;
    vector<Edge> edges = {{0, 1, 4}, {0, 2, 5}, {1, 2, -3}, {2, 3, 4}};
    bellmanFord(edges, V, 0);
    return 0;
}

📚 Real-world Applications:

  • Currency arbitrage detection
  • Distance-vector routing (e.g., RIP protocol)
  • Optimizing cost with possible losses

✅ Pros & ⚠️ Cons:

  • ✅ Pros: Works with negative weights, simple to implement
  • ⚠️ Cons: Slower than Dijkstra on dense graphs
