↪️ Insertion Sort Algorithm

A simple and intuitive algorithm ideal for small or nearly sorted datasets.

📘 What is Insertion Sort?

Insertion Sort works similarly to how we sort playing cards in our hands. It builds the final sorted array one item at a time by inserting elements into their correct position.

📍 Applications:

  • Great for small data sets
  • Efficient for nearly sorted arrays
  • Used in teaching basic sorting concepts

🐍 Python Example


def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

arr = [9, 5, 1, 4, 3]
insertion_sort(arr)
print("Sorted array:", arr)

💻 C++ Example


#include <iostream>
using namespace std;

void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

int main() {
    int arr[] = {9, 5, 1, 4, 3};
    int n = sizeof(arr)/sizeof(arr[0]);
    insertionSort(arr, n);
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++)
        cout << arr[i] << " ";
    return 0;
}

📈 Time & Space Complexity

  • Best Case: O(n) [Already Sorted]
  • Average Case: O(n²)
  • Worst Case: O(n²)
  • Space: O(1) In-place

✅ Advantages

  • Simple to implement
  • No extra memory required
  • Fast for nearly sorted data

⚠️ Limitations

  • Not efficient for large datasets
  • O(n²) in worst cases
