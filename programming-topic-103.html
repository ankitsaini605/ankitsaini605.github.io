Operators in Programming

Learn how to perform operations using arithmetic, logical, relational, and more types of operators.

Dive into Logic!

🧮 What are Operators?

Operators are symbols used to perform operations on variables and values. They act as the building blocks of logical and mathematical programming expressions.

🧠 Types of Operators:

  • Arithmetic Operators — `+`, `-`, `*`, `/`, `%`
  • Relational (Comparison) Operators — `==`, `!=`, `>`, `<`, `>=`, `<=`
  • Logical Operators — `and`, `or`, `not` (Python); `&&`, `||`, `!` (C++)
  • Assignment Operators — `=`, `+=`, `-=`, `*=`, `/=`
  • Bitwise Operators — `&`, `|`, `^`, `~`, `<<`, `>>`

✨ Arithmetic Operators Example (Python)


a = 10
b = 3
print(a + b)  # Addition
print(a - b)  # Subtraction
print(a * b)  # Multiplication
print(a / b)  # Division
print(a % b)  # Modulus

🔍 Relational Operators


x = 5
y = 10
print(x > y)   # False
print(x <= y)  # True

💡 Logical Operators Example


a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

🎯 Bitwise Magic

Bitwise operators work on bits and perform bit-by-bit operations.


a = 5  # 0101
b = 3  # 0011
print(a & b)  # 0001 -> 1
print(a | b)  # 0111 -> 7
print(a ^ b)  # 0110 -> 6
