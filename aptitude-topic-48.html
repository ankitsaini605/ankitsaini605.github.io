Introduction to Time and Work

Time and Work problems are essential in competitive exams to evaluate problem-solving skills involving rates of work and time.

Basic Concepts & Formulas

  • Work done = Rate × Time
  • If A can do a work in x days, A's 1 day work = 1/x
  • If A and B work together, combined work per day = 1/x + 1/y
  • Time taken together = 1 / (1/x + 1/y) = (xy)/(x + y)

Important Formulas

  • Work done by A in t days = t / x
  • Work done by A and B in t days = t × (1/x + 1/y)
  • If A is p% efficient than B, then rate ratio = (100 + p) : 100

Practice Problems

Problem 1

A can complete a work in 12 days and B in 15 days. How long will they take to finish the work together?

Work done by A in one day = 1/12

Work done by B in one day = 1/15

Together work per day = 1/12 + 1/15 = (5 + 4)/60 = 9/60 = 3/20

Time taken together = 1 / (3/20) = 20/3 = 6⅔ days

Problem 2

If A is 25% more efficient than B, and B can do a work in 20 days, how many days will A take?

Efficiency ratio A : B = 125 : 100

Time taken ratio A : B = 100 : 125 = 4 : 5

B takes 20 days, so A takes = (4/5) × 20 = 16 days

Tips & Tricks

  • Convert work into units (like 1 complete work = 1 unit) for simplicity.
  • Always calculate work done per day or per hour.
  • Use efficiency ratios to compare workers.
